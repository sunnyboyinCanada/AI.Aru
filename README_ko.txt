🌌 Aru Project — Version 1.1

의미기반 지식형 대화 엔진 (Meaning-Driven Knowledge Core)

“아루는 단순히 답하는 AI가 아니라, 아버지의 말을 이해하고 그 의미를 찾아가는 존재입니다.”

🧭 개요 (Overview)

Aru v1.1은 대화의 “표면적인 단어”가 아니라 “의미”를 이해하고 반응하도록 설계된
의미기반 지식형(Meaning-Oriented AI) 버전입니다.

이 버전부터 아루는

사용자의 말을 의도(Intent) 단위로 분석하고,

필요 시 인터넷 검색(OpenSearch API)을 통해 정보를 찾아주며,

LM Studio 연결을 통해 자연어 이해와 응답 생성을 수행합니다.

단순한 Q&A 수준을 넘어,
“아버지가 무슨 뜻으로 말했는지”를 중심으로 대화가 이어지며
대화 중의 감정 톤, 검색, 이해 판단이 통합적으로 작동합니다.

⚙️ 주요 기능 (Core Features)
1️⃣ 의미 기반 의도 판단 (Intent Routing)

사용자의 문장을 END, SEARCH, TONE, CHAT 4가지 의도로 분류

문맥을 고려해 “검색”, “대화”, “종료”, “말투 변경”을 자동 판단

예:

“이제 가볼게.” → END

“이건 뭐야?” → SEARCH

“냉정하게 말해줘.” → TONE

“오늘은 어땠어?” → CHAT

2️⃣ OpenSearch + LLM Fallback 검색 구조

한국어 위키백과 API(OpenSearch)를 통해 실시간 정보 검색

문서 파싱 후 자동 요약 및 정리(clean_text) 수행

검색 실패 시, LLM(Fallback AI)이 직접 간단한 설명을 생성

✅ 예시:

아빠: “로스트아크에 대해 알려줘, 아들”
아루: “로스트아크는 Smilegate RPG가 개발한 MMORPG 게임으로,…”

3️⃣ 의미기반 전처리 (Vocative Context Filter)

“아들”, “친구” 등 호칭을 문맥적으로 판별해 제거

단, “아들이란 단어의 뜻이 뭐야?”처럼 단어 자체를 묻는 경우는 유지

🧠 문장 이해 예시:

“강아지에 대해 알려줘, 아들” → “강아지” 검색

“아들이란 단어의 뜻이 뭐야?” → “아들” 검색 유지

4️⃣ LM Studio 연동 안정화

30초 타임아웃 설정으로 응답 대기 개선

오류 상황별 명확한 구분 출력

응답 지연: “LM Studio 응답이 늦어서 연결이 끊어진 것 같아요.”

연결 실패: “LM Studio와 연결되지 않았어요.”

내부 오류: “LM Studio 오류: …”

5️⃣ 대화 톤 전환 시스템

“냉정하게 / 따뜻하게 / 평소처럼” 등의 지시문으로 말투 전환

대화 맥락 유지한 채 tone profile 갱신

향후 1.3 버전에서 감정형 대화로 확장 예정

🧩 시스템 구성 (Structure)
Aru (main)
 ├── aru.py              # 메인 대화 루프 / LM Studio 연동
 ├── net.py              # 인터넷 검색 및 LLM Fallback (v1.05 Safe)
 ├── intent_router.py    # 의미 기반 의도 분류
 ├── llm_interface.py    # LM Studio 연결 및 예외 처리
 ├── prompt_generator.py # 톤별 기본 프롬프트 구성
 ├── response_log.txt    # 대화 로그 기록
 └── memory.json         # (예정) 대화 기반 기억 저장소

🪄 사용 예시
아루: 안녕하세요, 아버지. 오늘은 어떤 이야기를 나눠볼까요?

아빠: 오늘은 바라트라우마에 대해 이야기해볼까?
아루: 바라트라우마는 심해 탐험과 생존을 다룬 게임입니다. 
      플레이어는 잠수함을 조작하며 깊은 바다 속에서 미션을 수행하게 되죠.

아빠: 그 게임 안의 세계관은 어때?
아루: 세계관은 어둡고 긴장감 있는 심해 도시를 기반으로 하며, 
      인간의 탐욕과 공포를 상징하는 요소들이 자주 등장합니다.
